# User Guide

## Lock State
- App starts in locked mode (read-only).
- Click **Unlock Editing** in top bar to enable edits.

## Search
- Use the search box for FTS results.
- Matches are grouped by chapter; highlighted snippets appear below results.

## Edit Terms
- Unlock editing.
- Use `New Term`, fill fields, maintain synonyms, edit the annotation text, assign chapters, then `Save`.
- Use `Delete` to remove current term.

## Power-User Mode
- `Ctrl+K`: command palette.
- `Ctrl+N`: new term.
- `Ctrl+S`: save.
- `Ctrl+F`: focus search.
- `Ctrl+L`: toggle lock.
