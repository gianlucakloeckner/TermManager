# Editing Guide

There is no role-based login. Editing is controlled by the lock in the top bar.

## Edit Workflow
1. Click **Unlock Editing**.
2. Create or select a term.
3. Edit fields, synonyms, annotation text, chapters, and image.
4. Click **Save**.
5. Review changes in **History**.

## Chapter Management
- Use **Manage Chapters** to add/edit/delete chapters and visibility state.

## Data Operations
- **Import** supports JSON/CSV/Excel.
- **Export** writes JSON/CSV/Excel.
